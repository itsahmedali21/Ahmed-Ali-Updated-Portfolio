about

Code, Webflow, search, and a good prompt

I'm Ahmed Ali, a frontend developer based in Pakistan who also builds in Webflow, handles SEO, and leans heavily on AI-assisted ("vibe coding") workflows — four different routes to the same result: a fast site that's built right and easy to find.

On the code side, I work in HTML, CSS and JavaScript, with React when a project calls for it — and a good share of that is vibe-coded with tools like Claude and Cursor, going from idea to working build fast. On the Webflow side, I build CMS-driven sites clients can update without touching a line of code. On the SEO side, I run technical audits, fix Core Web Vitals, and turn keyword research into a real content and internal-linking plan.

Most projects start as a Figma file — or just a prompt — and end as a live site that ranks, not just one that looks good in a portfolio screenshot.
